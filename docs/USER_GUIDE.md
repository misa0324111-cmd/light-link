# LIGHT LINK ユーザー向け使い方

## 1. AIマッチング診断

1. `/matching` を開く
2. 質問に答える
3. 相性スコア順に候補が表示される
4. 気になる候補の詳細を見る
5. お気に入りまたは予約へ進む

## 2. AIチャット相談

1. `/chat` を開く
2. 希望条件を文章で入力
3. AIがおすすめ候補を表示
4. 詳細またはLINE相談へ進む

## 3. 好み学習

1. 候補詳細を見る
2. お気に入りに追加
3. `/preferences` で好み傾向を確認

## 4. LINE相談

1. `/line` を開く
2. LINE相談ボタンを押す
3. LINEで相談を続ける
