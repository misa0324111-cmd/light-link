# LIGHT LINK v4.1 SaaS化

## 追加内容

- `/store/signup`
- 店舗新規登録申請
- `/admin/store-applications`
- 店舗申請管理
- 承認/却下/保留
- `/store/onboarding`
- 店舗オンボーディング
- `/store/dashboard`
- SaaSダッシュボード
- `store_applications` テーブル

## 確認方法

1. `/store/signup` で店舗申請
2. `/admin/store-applications` で申請確認
3. 承認/却下/保留を操作
4. `/store/onboarding` で公開準備ステップ確認
5. `/store/dashboard` でKPI確認
