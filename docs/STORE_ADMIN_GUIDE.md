# LIGHT LINK 店舗管理者マニュアル

## 1. ログイン

1. `/login` を開く
2. 店舗管理者を選択
3. ログインする

## 2. キャスト管理

1. `/store-admin/talents`
2. キャスト登録
3. 編集
4. 非表示

## 3. 予約管理

1. `/store-admin/reservations`
2. 新規予約を確認
3. ステータスを変更

## 4. 画像管理

1. `/store-admin/media`
2. キャストを選択
3. 画像URLまたはStorageで画像登録

## 5. 通知

1. `/store-admin/notifications`
2. 未読通知を確認
3. 既読にする

## 6. プラン管理

1. `/billing`
2. プランを選択
3. Stripe連携時はCheckoutへ進む
