# LIGHT LINK 運営管理者マニュアル

## 1. 運営管理

`/admin` から運営管理機能に入ります。

## 2. KPI分析

`/admin/kpi`

確認できるもの:

- AI相談数
- LINEクリック
- 予約リクエスト
- 有料店舗数
- 集客ファネル

## 3. 本番公開チェック

`/admin/production`

環境変数の設定状況を確認します。

## 4. 通知テスト

`/admin/message-test`

メール/LINE通知テストを実行できます。

## 5. リリース確認

`/release`

全機能へのリンクを確認できます。
