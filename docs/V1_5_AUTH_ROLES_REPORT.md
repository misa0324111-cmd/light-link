# LIGHT LINK v1.5 ログイン・権限管理

## 追加内容

- デモログイン
- 役割選択
- 一般ユーザー / 店舗管理者 / 運営管理者
- AuthGate
- UserStatus
- 店舗管理画面の権限制御
- 運営管理画面の権限制御
- マイページ `/role`

## 確認方法

1. `/login` を開く
2. 店舗管理者でログイン
3. `/store-admin` が見られる
4. `/admin` は権限なしになる
5. 運営管理者でログイン
6. `/admin` が見られる
