- 通知センター
- `/store-admin/notifications`
- `/admin/notifications`

- `/api/notifications/reservation`
- 新規予約時の通知生成
- 既読管理

## 確認方法

1. `/reservation/new?talent=t1` で予約送信
2. `/store-admin/notifications` を開く
3. 新規予約通知が表示される
4. 既読にできる
